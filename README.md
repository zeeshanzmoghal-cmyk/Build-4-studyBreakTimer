# Project Name: studyBreakTimer

Study break Timer is an effective tool designed to help you stay focused while studying and help you study effectively. When your tired of some time of focused study, simply pull up a Study Break Timer window, chose either 5 min break, 10 min break or 15 min break, and then sit and relax by the computer and the program itself will let you know when break time is done. Even though its called "Study" Break Timer, you don't just have to use it for studying, you can use it while working, or just any task aswell.

**How to use**
*Option 1:*
  - Download the .exe file below and enjoy 😊

*Option 2:*
  - Copy repository URL
  - Clone repository in Microsoft Visual Studio (2019 or later)
  - Open project in Microsoft Visual Studio
  - Restore Dependencies (if necessary)
  - Build and run
  - Enjoy 😊
